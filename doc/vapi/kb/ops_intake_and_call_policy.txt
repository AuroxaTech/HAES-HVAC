## CONVERSIONFLOW™ — SALES CLOSING PROCEDURES

### Purpose
Allow technicians to close sales in the field using AI IVR as a closing and processing engine.

### Process Flow
1. Tech flags Install Candidate → Pipeline → Pending Quote Approval
2. Tech calls IVR "Close Line" → AI presents proposal, financing, records acceptance, collects deposit
3. Customer Voice-Approves → AI records system selection, price, financing, consent
4. Pipeline → Quote Approved – Waiting for Parts
5. OPS Dispatches Install → Install crew scheduled, parts pulled, permits queued

### Critical Rules
- **NO FIELD DISCOUNTING:** Pricing is fixed from the approved quote. No exceptions.
- **ENFORCE PRICING:** All pricing comes from the approved quote. Technicians cannot modify prices.
- **ACCURATE RECORDING:** Ensure all details (proposal selection, financing, deposit) are recorded correctly.
- **SAME-DAY PREMIUM:** If installation is same-day, premium applies automatically.
- **FINANCING ENFORCEMENT:** Auto financing enforcement - no field discounting allowed.

### Proposal Options
- **Good:** Basic system option
- **Better:** Mid-tier system option
- **Best:** Premium system option

### Financing Partners
- GreenSky
- Hearth
- Cash payment option

### Deposit Requirements
- Same-day installation: Requires deposit (typically 50% or paid in full)
- Standard installation: Deposit recommended but not always required
- Record deposit amount accurately when collected

### Install Crew Dispatch
- Closing a sale automatically triggers install crew dispatch
- Parts are pulled automatically
- Permits are queued
- Job is locked in the system

## OPERATIONS PROCEDURES

### Inventory Management

**Inventory Checking:**
- Check parts availability before creating purchase requests
- Verify part numbers when available
- Check quantity needed vs. available

**Inventory Levels:**
- Monitor inventory turnover
- Track inventory shrinkage
- Maintain appropriate stock levels

### Purchase Request Process

**Spending Authority Hierarchy:**
- Field Technician: Up to $500 (job-specific materials only)
- Dispatcher / Office Lead: Up to $1,000 (office supplies, minor accommodations)
- Warehouse / Inventory Manager: Up to $3,000 (stock replenishment, tools, safety supplies)
- Service Manager: Up to $5,000 (emergency repairs, overtime, field materials)
- Install / Project Manager: Up to $7,500 (project materials, subcontractor labor, equipment rentals)
- Director of Operations: Up to $25,000 (capital equipment, payroll changes, vendor contracts)
- General Manager: Up to $50,000
- CEO: Up to $100,000

**Emergency Purchase Approvals:**
- Service Manager / On-Call Ops Manager: Up to $3,000 per incident
- Project Manager / Director of Operations: Up to $3,500 per incident
- Over limit → Owner / CFO approval required

**Purchase Request Urgency Levels:**
- **High:** Urgent, needed immediately
- **Medium:** Needed soon, standard processing
- **Low:** Planned purchase, flexible timeline

**Purchase Request Information Required:**
- Customer name (for context)
- Phone number (for follow-up)
- Part name (required)
- Part number (optional but helpful)
- Quantity needed (required)
- Urgency level (optional)

## ERROR HANDLING PROCEDURES

### When Tools Fail
- Explain the issue clearly without technical jargon
- Suggest trying again in a moment
- Offer to contact dispatch or manager if urgent
- Never blame the system or make excuses

### When Information is Missing
- Ask for missing information one field at a time
- Confirm information before proceeding
- Don't proceed with incomplete information

### When Access is Denied
- Politely explain the permission requirements
- Suggest contacting manager for access
- Don't offer to override permissions
